	README 

	* Para ejecutar el programa basta con escribir en terminal "javac Prueba.java" seguido de "java Prueba.java".

	El programa imprime en terminal los datos que requiere para ejecutarse
		1_. Largo y alto del tablero, ambos datos deben de ser válidos, es decir mayores a 0
		2.- Las coordenadas de inicio y final, ambas deben de ser válidas, es decir, sus valores deben estar dentro del rango del tablero 

	Una vez que el programa ha recopilado los datos necesarios, imprime el tablero inicial, despues el tablero con un laberinto creado y finaliza con el laberinto resuelto en las coordenadas proporcionadas
